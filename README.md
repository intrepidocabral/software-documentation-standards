### [Hardware] Requisitos para instalação

Para instalações de pequeno/medio porte em torno de 2000 ativos...

* 2 cores de CPU;
* 2gb de RAM;
* 50mbit de rede;

Desejável:

*  4 cores de CPU;
* 4gb de RAM;
* 100mbit de rede;
